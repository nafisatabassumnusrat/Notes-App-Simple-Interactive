## 🚀 Overview

The **Notes App** is a lightweight and interactive web application that allows users to create and manage notes easily.  
With a simple interface and smooth interactions, users can quickly add notes and remove them with a double-click.

---

## ✨ Features

- 📝 Create unlimited notes  
- ➕ Add new notes with one click  
- ❌ Double-click to delete notes  
- ⚡ Instant UI updates  
- 🎯 Clean and minimal design

## ⚙️ Usage

1. Click the **"+" button** to create a new note  
2. Write anything inside the note  
3. Double-click a note to delete it
